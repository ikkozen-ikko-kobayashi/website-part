# サポート / お問い合わせ

本アプリケーション「ひと休み」（英字開発名: ZenHRV）に関するご不明点・不具合報告・改善要望などがありましたら、以下のメールアドレスまでご連絡ください。

**メールアドレス:** [ikko.kobayashi@ikkozen.com](mailto:ikko.kobayashi@ikkozen.com)

## 返信ポリシー

- 原則、営業日 3 日以内を目安に回答します。
- 内容によっては調査工数確保のため追加の時間をいただく場合があります。

## ご連絡時にあると助かる情報（任意）

- 発生日時 / 時刻
- 端末機種 / OS バージョン
- アプリバージョン / TestFlight ビルド番号
- 画面遷移手順（再現ステップ）
- もし表示された場合のエラーメッセージ

## よくあるお問い合わせの例

| 区分     | 例                                    | 対応                               |
| -------- | ------------------------------------- | ---------------------------------- |
| 機能要望 | HRV 推移の週次グラフがほしい          | 今後の改善候補として検討します     |
| 不具合   | 測定終了後に画面が固まる              | 再現手順と端末情報をご連絡ください |
| データ   | 過去 1 か月の記録をまとめて出力したい | エクスポート機能の拡張を検討中です |

## 免責

本アプリは医療診断を目的としたものではなく、ヘルスケア指標・休息サポートを補助的に提供するものです。緊急の健康・医療上の懸念がある場合は、速やかに専門医へ相談してください。

## プライバシー

個人情報や測定データの取り扱いについては別途掲載のプライバシーポリシーをご参照ください。
